Row System Rules

Sizing and spacing

  • Use one grid per row system with consistent gap values.
  • Use a shared minimum card width to avoid narrow columns.
  • Keep card padding consistent across card types.
  • Align cards to stretch height within a row.

Media rules

  • If any card in a row has no image, remove the media slot from all cards in that row.
  • Keep aspect ratios consistent (16:9 or 4:3) across image rows.
  • Use card-grid--text-only on the grid to suppress all media slots at once.
  • Never mix image and no-image cards in the same row.

Content rhythm

  • Clamp titles and summaries to keep rows even.
  • Place actions at the bottom to align buttons.
  • Use consistent metadata styling for type and timing.
  • Avoid mixing more than two action styles per row.

CSS Snippet

Copy and paste into your CSS

.card-grid {
    display: grid;
    gap: 24px;
    align-items: stretch;
}

.card-grid--auto {
    grid-template-columns: repeat(auto-fit, minmax(260px, 1fr));
}

.card-grid--fixed {
    grid-template-columns: repeat(4, minmax(0, 1fr));
}

/* Add to any row where one or more cards lack an image */
.card-grid--text-only .card-media {
    display: none;
}

.card {
    display: flex;
    flex-direction: column;
    background: var(--surface-bright);
    border: 1px solid var(--outline-variant);
    border-radius: 12px;
    overflow: hidden;
}

.card-media {
    aspect-ratio: 16 / 9;
    background: var(--surface-variant);
}

.card-actions {
    margin-top: auto;
    display: flex;
    gap: 12px;
}
