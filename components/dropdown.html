Usage Guidelines

When to Use

  • When users need to select from a predefined list of options
  • When space is limited and you need to conserve vertical space
  • When the number of options is between 3-15 items
  • When the options are mutually exclusive

When Not to Use

  • When there are only 2 options (use radio buttons instead)
  • When there are more than 15 options (consider a searchable dropdown)
  • When users need to select multiple options (use checkboxes)
  • When the options are not mutually exclusive

Accessibility

  • Ensure proper keyboard navigation (Tab, Enter, Arrow keys)
  • Provide clear focus indicators
  • Use descriptive labels and placeholder text
  • Ensure sufficient color contrast for all states
  • Support screen readers with proper ARIA attributes
